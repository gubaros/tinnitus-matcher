# Tinnitus Matcher

This project is a web application that helps users match their tinnitus with a generated noise. Users can adjust the frequency of the noise using a slider to find the exact match.

## Features

- **Frequency Slider**: Adjust the frequency from 20 Hz to 20,000 Hz.
- **Audio Playback**: Real-time sound generation using the Web Audio API.
- **Start Button**: Initiate audio playback with a user gesture.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd tinnitus-matcher
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

1. Start the development server:
   ```bash
   npm start
   ```
2. Open your browser and go to `http://localhost:3000`.
3. Click the "Start Audio" button and use the slider to adjust the frequency.

## Technologies Used

- React
- TypeScript
- Web Audio API
